6TH SEMISTER COMPUTER SCIENCE

UNIT 1: Structure of HTML

  1. Structure of HTML: HTML (Hyper Text Markup Language) is a markup language used to create the structure and layout of web pages. An HTML document consists of two main sections: the HEAD and the BODY. The HEAD section contains information about the document, such as the title, character encoding, and linked stylesheets or scripts. The BODY section contains the actual content of the web page that is displayed in the browser.
  2. Physical Tags: Physical tags in HTML are used to define the visual appearance of text. Examples of physical tags include < b> for bold, < i> for italic, and < u> for underline. These tags modify the text's appearance without altering its meaning or structure.
  3. Logical Tags: Logical tags in HTML are used to structure and organize the content of a web page. They define the meaning and semantic structure of the text. Examples of logical tags include < h1> to < h6> for headings, < p> for paragraphs, and < ul> and < ol> for unordered and ordered lists, respectively.
    • Image tag: < img> tag is used to insert images into an HTML page. Marquee tag: < marquee> tag is used to create scrolling or moving text or images on a web page.
    • Table tag: < table> tag is used to create tables to organize data in rows and columns.
    • Frame tag: < frame> tag is used to divide a web page into multiple sections or frames.
    • Form elements: HTML forms allow users to interact with web pages by inputting data. Form elements include < input>, < select>, < textarea>, etc.
    • DHTML (Dynamic HTML): DHTML combines HTML, CSS, and JavaScript to create interactive and dynamic web pages.
    • Types of style sheets and selectors: Introduces different types of style sheets (inline, internal, external) and CSS selectors to apply styles to HTML elements.
    • CSS (Cascading Style Sheets): CSS is used to style and format the appearance of HTML elements.
    • CSS properties: Describes various CSS properties used to modify the style and layout of HTML elements.
    • Dynamic changes: Explains how to use JavaScript and CSS to dynamically change the content and appearance of web pages.

UNIT 2: Introduction to JavaScript

  • Introduction to JavaScript: JavaScript is a scripting language used to add interactivity and dynamic behavior to web pages.
  • Features of JavaScript: Discusses the features and capabilities of JavaScript, such as its ability to manipulate HTML elements, handle events, and perform calculations.
  • Variables and Constants: Covers the concepts of variables and constants in JavaScript, their declaration, assignment, and usage.
  • Types of dialogue boxes: Describes different dialogue boxes in JavaScript, such as alert(), prompt(), and confirm(), used to interact with users.
  • Operators: Explains various operators in JavaScript, including arithmetic, assignment, comparison, and logical operators.
  • Control Structures: Covers control structures like if statements, switch statements, and loops (for, while, do-while) to control the flow of program execution.
  • Functions: Introduces functions, reusable blocks of code, their declaration, parameters, return values, and function invocation.
  • Modules: Discusses the concept of modules or code organization and reusability in JavaScript.
  • User-Defined Function: Demonstrates how to create custom functions tailored to specific needs.
  • Global Function: Explains predefined global functions in JavaScript, such as parseInt(), parseFloat(), and setTimeout().
  • Recursion: Covers the concept of recursion, where a function calls itself to solve a problem.

UNIT 3: Arrays, Events, and Objects

  • Arrays: Explains arrays, which are used to store multiple values in a single variable.
  • Multi-Dimensional Array: Discusses multi-dimensional arrays, which are arrays with multiple dimensions or levels.
  • Passing Array Arguments to Function: Covers passing arrays as arguments to functions for processing.
  • Events: Describes events in JavaScript, such as button clicks, mouse movements, and key presses, and how to handle them.
  • Types of Events: Introduces various types of events, including mouse events, keyboard events, and form events.
  • Event Handling: Explains how to handle events in JavaScript using event handlers and event listeners.
  • Types of Objects: Discusses different types of objects in JavaScript, including built-in objects like Date, Math, and String, as well as user-defined objects.

UNIT 4: XML, XSL, DOM, and Ajax

  • XML Structure: Introduces XML (eXtensible Markup Language), which is used for storing and transporting data.
  • Components of XML: Discusses the components of XML, including elements, attributes, and namespaces.
  • XSL (Extensible Stylesheet Language): Covers XSL, which is used to transform XML documents into other formats, such as HTML or PDF.
  • Document Type Definition (DTD): Explains DTD, which defines the structure and validation rules for an XML document.
  • DOM (Document Object Model): Describes the DOM, which represents the structure of an XML or HTML document as a tree-like structure that can be manipulated using scripting languages.
  • Ajax (Asynchronous JavaScript and XML): Introduces Ajax, a web development technique that allows for asynchronous data exchange between a web page and a server without requiring a full page reload.
  • Dojo Toolkit: Briefly covers the Dojo Toolkit, a JavaScript framework that provides tools and utilities for developing web applications.
  • These topics provide a comprehensive overview of the subject of Web Technologies and cover various aspects of web development using HTML, CSS, JavaScript, XML, and related technologies.
